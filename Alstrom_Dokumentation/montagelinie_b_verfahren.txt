Standard Operating Procedure: Montagelinie B

Anlaufprotokoll:
1. Hauptstromversorgung einschalten
2. Druckluftsystem prüfen (Idealdruck 4 Bar, min. 3, max 5)
3. Steuerungssoftware starten - Passwort: qur28XXm
4. Selbsttest durchführen
5. Referenzfahrt aller Achsen initiieren

Tägliche Prüfpunkte:
- Förderband auf Verschleiß prüfen
- Sensoren an Übergabestationen reinigen
- Sicherheitsschalter testen
- Notstopps auf Funktion prüfen

Produktwechsel:
Werkzeugwechsel an Station 3 und 7 erforderlich
Umstellung von CV-Software YOLOv8 auf YOLOv11 -> ultralytics muss upgedated werden, mit Befehl: pip install --update ultralytics
